- Navigasi sticky dan menu mobile.
- Hero section profesional.
- Ringkasan profil dan kontak.

## Menjalankan Secara Lokal

Tidak diperlukan proses instalasi.

Cukup buka:

```text
index.html
```

langsung melalui browser.

Alternatif menggunakan local server:

```bash
python -m http.server 8000
```

Kemudian buka:

```text
http://localhost:8000
```
